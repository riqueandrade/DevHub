Política de Privacidade

1. Coleta de Dados

A Dev Hub coleta informações necessárias para fornecer a melhor experiência possível aos usuários. Os dados coletados incluem:

  • Informações de registro (nome, email, senha)
  • Dados de uso da plataforma
  • Informações de pagamento (quando aplicável)
  • Dados técnicos (IP, navegador, dispositivo)

2. Uso dos Dados

Utilizamos seus dados para:

  • Fornecer acesso aos cursos e conteúdos
  • Melhorar nossos serviços
  • Enviar comunicações relevantes
  • Processar pagamentos
  • Prevenir fraudes e abusos

3. Proteção de Dados

A Dev Hub implementa medidas de segurança para proteger seus dados:

  • Criptografia de dados sensíveis
  • Acesso restrito a informações pessoais
  • Monitoramento regular de segurança
  • Backups periódicos

4. Compartilhamento de Dados

Seus dados podem ser compartilhados apenas:

  • Com seu consentimento explícito
  • Para cumprir obrigações legais
  • Com parceiros de processamento de pagamento
  • Para proteger nossos direitos ou propriedade

5. Seus Direitos

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar a exclusão de seus dados
  • Revogar consentimentos dados anteriormente

6. Cookies

Utilizamos cookies para melhorar sua experiência. Você pode controlar o uso de cookies através das configurações do seu navegador.

7. Contato

Para questões relacionadas à privacidade, entre em contato através do email: devhubsuporte@gmail.com
